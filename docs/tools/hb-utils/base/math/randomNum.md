# randomNum 指定范围的随意数

取最大值与最小值之间的随机数（包含最大值和最小值）。

## 基本用法
`
randomNum(min,max)
`

## 示例
```javascript
randomNum(100,200) // 125
```
## 参数


| 名称  | 备注 |  类型  | 是否必传 |可接受的参数值 | 默认值 |
| ----- | --- | ------ | --------|------------- | ----- |
| min | 最小值|Number|是 |- | -|
| max | 最大值| Number|是 | -|-|

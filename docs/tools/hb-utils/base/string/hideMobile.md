# hideMobile  手机号码脱敏

将手机号码中间四位替换成星号的函数，接受一个参数mobile为需要替换的手机号码。

函数将传入的手机号码进行替换，并返回替换后的字符串。

## 基本用法
`
hideMobile(mobile)
`

## 示例
```javascript
hideMobile(17672807215) //176****7215

```
## 参数


| 名称  | 备注 |  类型  | 是否必传 |可接受的参数值 | 默认值 |
| ----- | --- | ------ | --------|------------- | ----- |
| mobile | 需要隐藏的电话号码|String/Number|是 |- | -|
|

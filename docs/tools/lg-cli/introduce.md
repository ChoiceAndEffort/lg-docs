# 脚手架

## 简介
lg cli 是一个基于 node 开发的脚手架，是项目开发的标准工具：


## 使用

```sh
npm i -g @lg/lg-cli  //安装

lg-cli --version  //查看版本

```



## 属性

| 属性名称       |       描述       |
| -------------- | :--------------: |
| create-project | 快速创建模板项目 |
| add-module     | 快速创建标准模块 |

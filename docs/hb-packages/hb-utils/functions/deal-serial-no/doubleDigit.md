# doubleDigit 序号十位补零

小于10的数字处理为双位数。

## 基本用法
`
doubleDigit (index)
`

## 示例
```javascript
doubleDigit(0)// 01


```
## 参数



| 名称  | 备注 | 类型 | 是否必传| 可接受的参数值 | 默认值 |
|  ---  | ----| ---- | -------|------------- | ------- |
| index |传入需要转化的值|Number| 是  |-  | -|
